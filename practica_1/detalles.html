UniTools: Detalles de la página

El objetivo de UniTools es proporcionar una herramienta enfocada a las necesiades universitarias

de los estudiantes de Ingeniería informática. Para lograr este objetivo, UniTools quiere innovar

la forma de hacer proyectos basándose en cuatro funcionalidades que la hacen única.

Información adicional sobre cada una de estas funcionalidades pueden ser consultadas en el apartado "bocetos".

Tipos de usuarios

Usuarios no registrados: Tendran acceso limitado a la página. No tendrán un perfil creado, no podrán

subir código a la página, no podrán subir preguntas al foro ni mandar mensajes haciendo uso del servicio

de mensajería. Sí podrán utilizar las aplicaciones de la pestaña "herramientas"


Usuarios registrados: Tendrán acceso a todas las funciones "básicas" de la página, esto incluye: tener un perfil,

subir código a los repositorios, participar en el foro, enviar mensajes y usar las herramientas de la página.


Usuarios de pago/premium: Además de las funciones de los usuarios registrados, podrán crear repositorios privados

que serán cifrados con clave pública para mayor confidencialidad.

Gestión de Código

Como portal de gestión de código, UniTools implementará mecanismos de gestión de versiones para que

los miembros de un solo equipo puedan mantener su código único al día, así como mecanismos de privacidad

para que los grupos puedan crear repositorios privados cifrados con su clave pública.

Foro

En la web también habrá a disposición del usuario un foro en el que se podrán discutir sobre cualquier

temática relacionada con la facultad, cursos o asignaturas.

Mensajes

En la pensataña de mensajes el usuario podrá interactar con los demás miembros de la plataforma de forma simple,

tendremos un listado con los chats ya empezados con otros usuarios y la opción de empezar un nuevo chat.

Se dispondrá de una barra de búsqueda para encontrar de forma rápida el usuario con el que se quiere abrir un nuevo chat.

Herramientas

En UniTools también tendremos a disposición del usuario varias herramientas hechas con javascrip o php

en la que se podrán hacer conversiones de unidades, obtener hashes de archivos entre otros
